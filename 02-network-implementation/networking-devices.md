# Networking Devices

## Hub (Layer 1)
- Broadcasts everything
- No intelligence

## Switch (Layer 2)
- Uses MAC addresses
- Ethernet LAN switching
- Supports VLANs, STP

## Layer 3 Switch
- Routing + switching
- Inter-VLAN routing

## Router (Layer 3)
- IP-based routing
- NAT, gateways

## Firewall (Layer 3–7)
- Filters traffic
- Stateful / stateless / NGFW

## IDS / IPS
- IDS = detects
- IPS = blocks

## Load Balancer
- Distributes traffic
- Layer 4–7

## Proxy Server
- Filters/caches traffic
- Forward / reverse proxy

## Access Point
- Wireless bridge (Layer 2)

## Media Converter
- Copper ↔ Fiber (Layer 1)
