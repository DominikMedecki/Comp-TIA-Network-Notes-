# Wireless & WAN Technologies

## Wi-Fi (802.11)
- Managed by IEEE
- Wi-Fi Alliance ensures compatibility

## Cellular
### 4G LTE
- ~150–300 Mbps

### 5G
- 100 Mbps – multi-Gbps
- Low latency, IoT support

## Satellite
- High latency (~250ms+)
- Remote connectivity
- Starlink improves latency

## Ethernet
- IEEE 802.3 standard
- Copper + fiber
