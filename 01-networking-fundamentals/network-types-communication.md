# Network Communication Types

## Unicast
- One-to-one communication
- Used for web browsing, file transfers

## Multicast
- One-to-many (selected group)
- Streaming, routing updates

## Broadcast
- One-to-all in subnet
- ARP, IPv4 only (not IPv6)

## Anycast
- One-to-nearest of many
- DNS, CDN routing
