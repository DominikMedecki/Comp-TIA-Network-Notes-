# Ethernet & Cabling

## Ethernet Standards (802.3)
- Defines wired networking

Example:
1000BASE-T
- 1000 = speed (1 Gbps)
- BASE = baseband
- T = twisted pair

---

## Cable Types

### Twisted Pair
- UTP / STP
- Cat5e, Cat6, Cat6a

### Fiber Optic
- Single-mode: long distance (laser)
- Multimode: short distance (LED)

### Coaxial
- RG-6 (TV, internet)

### Twinax
- Short-distance high-speed (10Gbps)

---

## Connectors
- RJ45 → Ethernet
- RJ11 → Telephone
- SC / LC / ST → Fiber
- BNC → Coax
- F Connector → Cable TV

---

## Transceivers
- SFP / SFP+
- QSFP (4 channels)

---

## Fiber Advantages
- Long distance
- Immune to EMI
- High speed
