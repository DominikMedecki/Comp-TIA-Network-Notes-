# IPv4 & Binary Fundamentals

## Binary Basics
- 1 bit = 0 or 1
- 1 byte = 8 bits (octet)

## IPv4 Address
Example: 192.168.1.1
- Unique identifier
- Logical addressing (Layer 3)

## Subnet Mask
- Example: 255.255.255.0
- Defines network vs host portion

## Default Gateway
- Router IP on local subnet
- Used for external network access

## Loopback Address
- 127.0.0.1 → self-test

## Virtual IP (VIP)
- Not tied to physical NIC
- Used in load balancing/virtualization
