# OSI Model (CompTIA Network+)

## OSI Layers (Top → Bottom)
Application → HTTP, HTTPS, DNS, SMTP (user-facing apps)
Presentation → SSL/TLS, encryption, encoding, compression
Session → session management, checkpoints, RPC, NetBIOS
Transport → TCP/UDP, segmentation, ports
Network → IP addressing, routing, routers
Data Link → MAC addresses, Ethernet, switches
Physical → cables, signals, bits

## Layer Breakdown

### Layer 1 – Physical
- Bits, signals, cabling
- Fiber, copper, wireless
- Devices: hubs, media converters
- Issues: cable damage, attenuation, no link light

### Layer 2 – Data Link
- Frames, MAC addresses
- Switches, NICs
- Ethernet (802.3)
- LAN communication only

### Layer 3 – Network
- Packets, IP addresses
- Routers
- Routing between networks
- Fragmentation

### Layer 4 – Transport
- TCP (reliable) / UDP (fast)
- Segmentation, ports
- End-to-end communication

### Layer 5 – Session
- Session setup, maintenance, teardown
- RPC, NetBIOS, SIP

### Layer 6 – Presentation
- Data formatting
- Encryption (SSL/TLS)
- Compression, encoding

### Layer 7 – Application
- User services
- HTTP, HTTPS, DNS, FTP, SMTP, SSH
